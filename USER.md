# USER.md

## ユーザーについて

このプロジェクトのユーザーは、開発者本人です。

商用の一般配布アプリを最初から作るのではなく、まずは **自分が使うための軽量な動画確認・IN / OUT取得ツール** を作ることを目的としています。

## ユーザーの目的

Windows上で、主にH.264のmp4動画を軽快に再生しながら、IN / OUTの `hh:mm:ss:ff` を取得し、ffmpegで切り出すためのコマンドを作りたいです。

特に、一般的な動画プレーヤーでは不足しがちな以下を重視します。

- `hh:mm:ss:ff` のタイムコード表示
- FPS自動判別
- 左右キーでの1フレーム送り / 戻し
- `I` / `O` でのIN / OUT記録
- IN / OUTからffmpeg切り出しコマンドを生成してコピー
- 軽快な操作感
- シンプルなUI
- 動画確認に集中できる画面

## 背景

もともとはMac向けの軽量動画プレーヤーとして考えていました。

Mac版では、QuickTime Playerの軽さや見た目を参考にしつつ、標準機能では不足するタイムコード確認機能を追加する想定でした。

今回は、その考え方を **Windows向けに置き換えて進めたい** です。

そのため、Mac前提の技術やUIをそのまま使うのではなく、Windows 11向けに自然な実装方針へ置き換えてください。

## 対象OS

- Windows 11を主対象とします。
- Windows 10対応は必須ではありません。
- Microsoft Store配布は初期段階では不要です。
- 一般配布は想定せず、個人利用を優先します。
- まずはローカルで起動して使えることを優先します。

## 好み・重視点

### 軽量性

重いアプリにはしたくありません。

Electronのような重いフレームワークは、初期候補から外したいです。

### シンプルさ

プレイリストや複雑な設定画面は不要です。

動画を開いたらすぐ再生でき、必要な情報がすぐ確認できることを重視します。

### 実用性

見た目の完成度よりも、まずは実際に使えることを優先します。

特に以下が重要です。

- 動画が開ける
- H.264のmp4動画が再生できる
- タイムコードが表示される
- 左右キーでフレーム移動できる
- IN / OUTを記録できる
- ffmpegコマンドをコピーできる
- 操作が重くない

### 視認性

タイムコード表示は重要です。

数字が再生中に横揺れしないように、等幅数字または等幅フォントを使ってください。

候補:

- Cascadia Mono
- Consolas
- Segoe UIの等幅数字指定

## 操作イメージ

最低限、以下の操作ができればよいです。

- 動画ファイルを開く
- 動画をドラッグ＆ドロップで開く
- `Space` で再生 / 一時停止
- `Right Arrow` で1フレーム送り
- `Left Arrow` で1フレーム戻し
- `I` でINを設定
- `O` でOUTを設定
- `Esc` で選択中のINまたはOUTをクリア
- `Delete` でIN / OUTを両方クリア
- タイムコードを常時確認する
- IN / OUTが揃ったらffmpegコマンドをコピーする

## UIイメージ

## 最終的な理想

Windowsで動画ファイルを開き、軽快に再生しながら、タイムコードとフレーム位置を確認できる。

必要な位置でIN / OUTを設定し、再生中のソースファイル名から `source-cut.ext` の出力名を作ったffmpegコマンドをコピーできる。

多機能ではなくてもよいので、**軽い・見やすい・すぐ使える** ツールにしたいです。
